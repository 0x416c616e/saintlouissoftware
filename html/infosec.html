Infosec

Information security is really fascinating. I have taught myself some security concepts already. Additionally, I will be taking more CMIS security classes at SIUE later. I am also beginning independent study through Virtual Hacking Labs, which I will document on my blog. RSS is kind of old-fashioned, so I will post updates about it on social media instead. Follow me on Twitter if you want to be notified when I write a new security post.

To be a good software developer, you have to be conscious of information security, even if you're not in a security-focused role. You can partially blame hackers, but a lot of the onus is on the person developing the software.
